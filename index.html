Sobre Mim

Eu sou Rick Simões Rabelo, e atualmente estou cursando área de Desenvolvimento de Sistemas no SENAI - CETAF AJU. Comecei a me interessar pela área ainda no primeiro ano do Ensino Médio integrado, porém, só comecei a estudar definitivamente no segundo ano. Neilton Barreto, meu lecionador, é o responsável por nos instruir desde o início, nos ensinando todos os conceitos de lógica da programação, elaborando atividades e avaliações teóricas e práticas para o nosso desenvolvimento nesse âmbito. Atualmente estamos no último semestre do curso, no momento na área de Programação de Aplicativos, com ênfase em criação de sites usando HTML, CSS e JavaScript. O módulo possui uma carga horária total de 160 horas. Abaixo seguem algumas atividades de projetos elaborados ao decorrer do ano:

Projetos

Projeto 1

Como uma das últimas atividades elaboradas e projetos iniciais com a inclusão do JS até então, pudemos criar um script que operava e executava as quatro operações básicas (soma, subtração, multiplicação, adição). Com isso, elaborei um modelo de uma calculadora comum, ao qual atendia os requisitos solicitados da atividade. Utilizamos HTML, CSS e JavaScript.

Projeto 2

Nesse projeto, pude compreender como funciona a construção e estilização de um site utilizando HTML. Sendo essa uma das primeiras atividades elaboradas do assunto, ficamos encarregados de criar o nosso próprio Perfil Social, onde pude identificar elementos importantes para a inclusão e estilização de um site. No mais, fiquei muito contente com o resultado final.

Projeto 3

O projeto da literatura de cordel foi muito interessante de ser feito. Com o objetivo de transformar a poesia e levá-la para o nosso próprio site, fomos induzidos a criá-lo, com elementos já disponibilizados em uma pasta para a estilização. Usando HTML e CSS, criei um site com uma poesia inclusa, onde utilizei nas imagens inclusas às estrofes o efeito Parallax para mantê-las estáticas mesmo utilizando o scroll para visualização completa. Além disso, incluímos fontes específicas para a elaboração do projeto do cordel.
